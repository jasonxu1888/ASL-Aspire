For installing packages...

Make sure root dir is SP2023

Commands:
python3 -m env .venv
.venv\Scripts\activate
pip install -r requirements.txt

Note:
There are very likely unnecessary packages/libraries in the requirements.txt, but if it works it works. Cleaning up can be done, but good luck...


To run the game...

Make sure virtual environment is activated and root dir is SP2023

Commands:
python game.py
